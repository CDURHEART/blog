## 1. 搜索
### in:xxx sss sss

in:xxx sss sss => 检索xxx中有sss的仓库

| 格式    | 限定范围        |
| ----------- | ----------- |
| in:name   | 仓库名称中 |
| in:description  | 摘要中  |
| in:readme   | README.md中  |
